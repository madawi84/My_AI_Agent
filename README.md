# My AI Agent

A **multi-agent AI research system** designed to answer user questions through a structured reasoning pipeline.

The system decomposes a user query into smaller tasks and processes them through specialised agents responsible for **planning, research, analysis, fact-checking, and final writing**.

This project demonstrates how **AI agents can collaborate to produce more reliable and structured responses**.

---

## Project Overview

Instead of using a single large language model directly, this system uses a **multi-agent architecture** where each agent performs a specific role in the reasoning process.

This approach improves:

- reasoning structure
- response quality
- modularity
- traceability

---

## Architecture

The system follows a **pipeline of specialised agents**.

```
User Query
    │
    ▼
Planner Agent
    │
    ▼
Researcher Agent(s)
    │
    ▼
Analyst Agent
    │
    ▼
Fact-Checker Agent
    │
    ▼
Writer Agent
    │
    ▼
Final Response
```

Each agent contributes to producing the final answer.

---

## Agent Roles

### 1. Planner Agent
Responsible for understanding the user query and breaking it into smaller research tasks.

Example output:

- Define the concept
- Explain how it works
- Provide real-world applications

---

### 2. Researcher Agent
Retrieves information needed to answer each subtask.

Depending on the complexity of the query, the system may create **multiple researcher agents**.

---

### 3. Analyst Agent
Processes the research results and extracts the key insights required to answer the question.

---

### 4. Fact-Checker Agent
Validates the information generated by the system to reduce hallucinations and improve reliability.

---

### 5. Writer Agent
Combines all verified information and produces a clear, structured final response.

---

## Example Workflow

Example user query:

```
What is quantum computing?
```

Execution steps:

1. The **Planner Agent** decomposes the query into subtasks.
2. The **Researcher Agent(s)** gather relevant information.
3. The **Analyst Agent** synthesises the research findings.
4. The **Fact-Checker Agent** validates the information.
5. The **Writer Agent** generates the final answer.

## Installation

Clone the repository:

```bash
git clone https://github.com/madawi84/My_AI_Agent.git
cd My_AI_Agent
```

Create a virtual environment:

```bash
uv venv
```

Activate the environment.

### Windows

```bash
.venv\Scripts\activate
```

### Mac / Linux

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
uv pip install -e .
```

---

## Environment Setup

Create a `.env` file in the root directory.

## Running the Agent

Run the research pipeline with:

```bash
uv run python -m src.main research "What is quantum computing?"
```

Example output metadata:

```
Metadata:

researcher_traces
analyst_trace
fact_checker_trace
writer_trace
```

These traces show how each agent contributes to generating the final answer.
